# Setup

- Clone the repo `git clone https://github.com/maruf99/robotman.git`
- Rename `.env.example` to `.env` and fill it out
- Run `npm i`
- Install PM2 `npm i -g pm2`
- Run `npm run build`
- Run `npm run start` to start the bot
